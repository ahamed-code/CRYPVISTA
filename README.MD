# Cryptocurrency Price Prediction App

This Streamlit web application predicts cryptocurrency prices and visualizes the predictions in real-time.

## Features
- Login functionality with email and password.
- Multiselect option to choose cryptocurrencies.
- Visualization of actual and predicted prices using line charts.
- Summary of actual and predicted mean prices with differences.
- Sidebar input for setting desired prices with real-time notification.

## Installation
1. Download Zip file:

extract it to your desired location.


2. Install the required dependencies:

```
pip install streamlit 
pip install pandas 
pip install matplotlib 
pip install scikit-learn
```

3. Run the Streamlit app:

```
streamlit run crypto.py
```

## Usage
- Upon running the app, enter your email and password to log in.
- Select cryptocurrencies from the multiselect dropdown.
- Visualize actual and predicted prices for the selected cryptocurrencies.
- Set desired prices in the sidebar input for notifications.
